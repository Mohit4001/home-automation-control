##Automatic Fan Speed Controller (Temperature-Based)
# Project Description

This project demonstrates a temperature-based automatic fan speed controller using embedded system concepts.
The system adjusts fan speed automatically according to ambient temperature, improving energy efficiency and ensuring smooth operation compared to traditional ON/OFF control.

The project focuses on ADC-based sensing, PWM-based control, and sensor-to-actuator interfacing, making it suitable for embedded systems and automation learning.
